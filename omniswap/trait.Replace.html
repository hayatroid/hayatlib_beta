Expand description

Takes out the value from a reference, leaving another value you supplied.

This is a generalization of mem::replace that additionally supports Cell and RefCell.

§Usage

let mut x = 42;
assert_eq!(Replace::replace(&mut x, 84), 42);
assert_eq!(x, 84);
